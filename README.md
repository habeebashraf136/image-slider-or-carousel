## Image Slider / Carousel

A simple and responsive **Image Slider / Carousel** built using **HTML, CSS, and JavaScript**.  
This slider supports **Next/Previous buttons**, smooth transitions, and is easy to customize.

##  Features
-  Clean and modern design  
-  Fully responsive  
-  Manual slide navigation  
-  Smooth transitions  
-  Easy to integrate in any project
